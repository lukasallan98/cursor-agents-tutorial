# Cursor Agents Tutorial

Sandbox repository for learning and testing Cursor agent workflows.

## Description

This project demonstrates a basic Python project structure for working with Cursor agents. It includes a simple example that shows how to organize code with a main entry point and utility functions.

## Project Structure

```
cursor-agents-tutorial/
├── src/
│   ├── __init__.py      # Makes src a Python package
│   ├── main.py          # Main entry point script
│   └── utils.py         # Utility functions
└── README.md
```

## Requirements

- Python 3.x

## Getting Started

Run the project as a module from the project root:

```bash
python -m src.main
```
